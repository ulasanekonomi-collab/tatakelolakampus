# TatakelolaKampus

TatakelolaKampus is an AI-assisted organizational governance prototype for higher education institutions.

The project aims to help university leaders:
- interpret organizational dynamics,
- detect cultural and governance signals,
- identify convergence and divergence patterns,
- and support reflective institutional decision-making.

---

# Core Vision

TatakelolaKampus is designed as:

- organizational sensemaking system,
- governance co-pilot,
- organizational pulse engine,
- and reflective AI assistant.

The system focuses on:
- dialogue,
- organizational learning,
- cultural interpretation,
- and human-centered governance.

TatakelolaKampus is NOT designed as:
- surveillance system,
- employee monitoring tool,
- punitive governance engine,
- or automated managerial replacement.

---

The project is built upon several key concepts:

# Current Architecture

## Philosophy Layer
Humanistic and dialogical governance foundations.

## Ontology Layer
Organizational entities, relationships, and governance logic.

## Prompt Architecture
AI reasoning structure for organizational interpretation.

## Schema Layer
Standardized organizational interaction structures.

## Dataset Layer
Sample organizational interaction datasets.

## Backend Layer
FastAPI-based organizational analysis engine.

# Example Organizational Analysis

Input:

```json
{
  "narrative": "Senior lecturer questioned why policy discussions were not initiated through lecturer discussions."
}
```

Output:

```json
{
  "themes": [
    "participation"
  ],
  "emotional_signals": [
    "critical_reflection"

  "dynamics_classification": "constructive_divergence",
  "organizational_signals": [
    "participation_gap_signal"
  ],
  "governance_recommendations": [
    "increase participatory dialogue"
  ]
}
```
